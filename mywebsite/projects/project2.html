About Instagram Profile Page UI – Flutter Project

This project is a Flutter-based UI clone of the Instagram Profile Page. It replicates the layout and structure of a user's profile on Instagram,

focusing on clean design, responsiveness, and modern aesthetics.

Key Features

Profile Header

Displays user image, username, bio, and external website with a clean, Instagram-style layout.

User Stats

Shows post count, followers, and following numbers in a modern and bold layout.

Edit Profile Button

Functional and styled button for navigating to the profile edit screen.

Story Highlights

Horizontal scrollable row showing circular images and labels for user story highlights.

Post Grid

Grid layout that replicates Instagram's photo gallery using local or remote images.

Fully Responsive

Mobile-friendly design that adjusts to all screen sizes, including iPhone 15.
