About Kurd Shop

Kurd Shop is a feature-rich e-commerce application built with Flutter that provides users with an intuitive shopping experience. The app showcases products with beautiful animations, offers secure payment options, and includes admin functionality for managing inventory.

Designed with a focus on performance and user experience, Kurd Shop delivers native-like performance across both iOS and Android platforms while maintaining a single codebase.

Key Features

Responsive UI

Beautifully designed interface that adapts to all screen sizes with smooth animations and transitions.

Advanced Search

Powerful search functionality with filters, categories, and real-time suggestions.

Cart System

Intuitive cart management with quantity adjustment, wishlist, and saved for later options.

Secure Payments

Multiple payment options including credit cards, PayPal, and mobile money with end-to-end encryption.

Push Notifications

Instant order updates, promotions, and personalized recommendations through push notifications.

Admin Dashboard

Comprehensive admin panel for managing products, orders, customers, and viewing analytics.

Multi-language support

supporting multiple languages for managing products, orders, and customers globally.
